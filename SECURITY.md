# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in this application,
please report it privately:

- Email: **security@baliflowagency.com**
- Do not open a public GitHub issue for security reports.

Include what you found, the URL/endpoint involved, and steps to reproduce.
We aim to acknowledge reports within 72 hours.

## Scope

- Web application at `crm.baliflowagency.com` and its public pages
  (`/s`, `/b`, `/g`, `/m`, `/r`, `/rv`, `/u`, `/d`).
- API routes under `/api/`.

## Out of scope

- Denial of service / volumetric attacks.
- Social engineering of staff or venues using the platform.
- Third-party services (Supabase, Vercel, Meta, Stripe, PayPal) themselves.
